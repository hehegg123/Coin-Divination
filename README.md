# Coin Divination

一个无需安装依赖的本地网页原型，包含：

- 出生日期、时间、时区输入
- 简化版四柱推演
- 基于书籍主题提炼的规则包
- 六次古钱起卦
- 今日总运、分类运势、宜忌与建议

## 启动

```powershell
node server.mjs
```

然后访问 `http://localhost:4173`。

## 当前规则范围

- 年柱：以立春近似边界处理
- 月柱：以常见节令交接日近似处理
- 日柱：按 1900-01-01 为甲子日的 60 日循环推算
- 时柱：按日干起时公式推算
- 日主强弱：按月令、同党、耗泄、克制做近似量化
- 调候：加入《进阶》里十干调候与寒暖燥湿优先级思路
- 格局：区分格局用神与扶抑/调候辅助
- 今日关系：按流日/流月天干映射十神关系
- 喜忌：按强弱推导偏宜与偏忌五行
- 星宫同参：父母、婚姻、子女、财富、健康按十神与宫位并看
- 纳音：作辅助提示，不替代干支五行
- 卦象：使用六次古钱结果生成上下卦与动爻提示

## 规则落点

- [book-rules.js](./book-rules.js)：书籍规则摘要与启用范围
